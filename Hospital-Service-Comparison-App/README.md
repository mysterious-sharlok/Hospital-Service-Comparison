# Hospital Service Comparison App

A full-stack hospital comparison platform where users can:
- Search nearby hospitals
- Compare MRI/Xray/Blood Test prices
- View ratings and distance
- Book appointments
- View hospitals on maps

## Tech Stack
Frontend: React + Tailwind CSS
Backend: Node.js + Express
Database: MongoDB

## Features
- Authentication
- Hospital Search
- Appointment Booking
- Ratings
- Google Maps Integration
- Nearby Hospital Search
